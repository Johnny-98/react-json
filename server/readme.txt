Backend:
    Server Setup: Using Node.js with Express.js framework
    Websockets: communication between the server and clients
    In-Memory Storage: Maintain an in-memory data structure (e.g., JavaScript array or object) to store user details.

Libraries: 
    express - to create the server
    cors - in order to make the connection between frontend and backend 
    nodemon - restart server everytime i make changes 
    Socket.io - real-time bidirectional communication between the server and clients

How to start:
    npm start
